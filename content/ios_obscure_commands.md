# TCAM

## Forwarding Resources
```
show platform hardware capacity forwarding
```
## Total TCAM Usage
```
show platform hardware cef maximum-routes
```
## Fib Exception Status
```
show mls cef exception status
```
OR
```
show platform hardware cef exception status
```
